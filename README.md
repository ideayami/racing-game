# NEON APEX GP 🏁

疑似3D(Mode-7 / スーパーマリオカート風)のブラウザ・レーシングミニゲームです。
外部のゲームエンジンやライブラリは使わず、`<canvas>` と素の JavaScript だけで実装しています。

- 1コース・全3ラップのタイムアタック
- ベストタイムはブラウザの `localStorage` に自動保存
- キーボード操作 / タッチ操作の両対応（スマホでも遊べます）

## 遊び方

1. `index.html` をブラウザで開く（ダブルクリックでOK）
2. `Enter` キー、またはタップでスタート
3. 3ラップ走り切ったタイムを競います

### 操作方法

| 操作 | キー |
|---|---|
| 加速 | `↑` / `W` |
| 減速・後退 | `↓` / `S` |
| ステアリング | `←` `→` / `A` `D` |
| 一時停止・再開 | `Esc` |
| スタート・リトライ | `Enter` / 画面タップ |

コースの外（芝生）に出るとスピードが大きく落ちるので、カーブでは道からはみ出さないように！

## ファイル構成

```
racing-game/
├── index.html   # 画面構造・HUD・タイトル/ポーズ/結果画面
├── style.css    # レトロ・シンセウェーブ風のビジュアル
├── game.js      # 疑似3Dレンダリング・物理演算・ラップ判定など全ロジック
└── README.md
```

ビルドツールや `npm install` は不要です。静的ファイルのみで完結します。

## GitHub での公開手順

まだリポジトリを作っていない場合：

```bash
cd racing-game
git init
git add .
git commit -m "Initial commit: NEON APEX GP"
git branch -M main
git remote add origin <あなたのリポジトリのURL>
git push -u origin main
```

### GitHub Pages で公開する

1. GitHub のリポジトリページで **Settings → Pages** を開く
2. **Source** を `Deploy from a branch` に設定
3. ブランチを `main`、フォルダを `/ (root)` に設定して **Save**
4. 数分後、`https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます

## 仕組みメモ（疑似3Dのアルゴリズム）

コースは一定長さの「セグメント」の並びとして定義し、各セグメントが持つ `curve`（カーブ量）を
積み上げることでコース中心線の横方向オフセットを計算しています。

画面を横一列（スキャンライン）ごとに描画し、各行が「自車から何ワールド距離先を表しているか」を
逆算 → その距離でのコース中心線・道幅を透視変換してX座標・幅に変換、という流れです。
地平線に近い行ほど遠く（小さく）、画面下ほど近く（大きく）描かれることでカーブや奥行きが表現されます。
